date: 2026-02-02T11:31:06-08:00
draft: false
---

> **INFO** 动画速度修改为 0

```shell
defaults write com.apple.dock MSStagingManagerHideDelay -float 0
defaults write com.apple.dock MSStagingManagerShowDelay -float 0
killall Dock
```

> [!INFO] 恢复默认

```shell
defaults delete com.apple.dock MSStagingManagerHideDelay
defaults delete com.apple.dock MSStagingManagerShowDelay
killall Dock
```
